## ⭐ STYLE PREFIX — prepend to EVERY prompt

```
Friendly mobile-game cartoon style, soft rounded shapes, bold clean black
outlines, warm saturated colors, soft cel shading, slightly chunky cute
proportions, playful humorous mood, high quality game asset.
```

## 🔧 TECH RULES by type

- **Cards / trumps:** `single character centered, full body, 3/4 view, simple flat solid-color background (no scene), no text, no card frame, no border, vertical portrait, transparent PNG background`
- **Buildings:** `single building isolated, 3/4 top-down view, transparent background, no ground scene, no text`
- **Duelists:** `seated character, waist-up, facing camera, simple background, no text`
- **Icons / currency:** `single icon, centered, transparent background, flat, clean silhouette, app-icon style, no text`
- **Screen backgrounds:** `wide landscape background, no characters, no UI, no text, calm center area for UI overlay`

## 🚫 NEGATIVES — append to EVERY prompt

```
No text, no letters, no watermark, no signature, no logo, no UI mockup, no
buttons, no card frame or border, no grid, no collage, no multiple variations
in one image — only ONE single subject per image. No drop shadow on the
background, no photographic realism.
```

## How to feed this to ChatGPT

1. **Paste the block above** as your first message in a **new chat**. ChatGPT will confirm it understands the style.
2. Say **`generate #1`** → it makes the first image. This becomes your **style anchor**.
3. If you like it, continue: **`generate #2`**, **`generate #3`**, … Each new image will match because the style lives in the chat context.
4. If an image drifts off-style, say: **`redo in the exact same style as image #1`**.
5. Keep all cards in one chat, all buildings in another chat, etc. — shorter chats stay more consistent than one giant 52-image chat.
6. For 3 building levels: after the base, say **`now the bigger more detailed level 2 version`**, then **`now the grand maxed-out glowing level 3 version`**.

## ⚡ Can I generate all 52 at once?

Short answer: **not as 52 separate clean files from the chat UI.** ChatGPT renders **one image per message** — that's a hard limit of the chat interface. But there are 3 ways to make it much faster than typing 52 times:

### Option A — Contact-sheet trick (fastest in the chat, lower quality)
Ask for a **grid** in one image, then crop it yourself afterwards:

```
Generate ONE image: a 3x4 grid of 12 separate card characters, evenly spaced,
each on its own flat background tile, in the STYLE above. The 12 characters:
<paste the 12 card descriptions>. Label nothing. Keep all 12 in the same style.
```

- ✅ 12 characters in one generation, instantly consistent (same image = same style).
- ❌ Each ends up ~300px after cropping (low res), characters may bleed together, you must cut them up in an image editor. Good for a **quick draft / style test**, not final assets.

### Option B — API batch script (true "all at once", recommended for final)
The OpenAI **Images API** (`gpt-image-1`) lets a script loop over all 52 prompts and save 52 full-res PNGs automatically — no manual typing. This needs an **API key** (platform.openai.com, billed per image ~$0.04–0.19 each → ~$2–10 for all 52). Note: this is **separate from your ChatGPT Plus subscription** — Plus does not include API credits.

If you want, I can generate a ready-to-run script (`design/generate_art.py`) that reads the prompt list and writes every asset straight into `assets/` with the right filenames. Just say the word and tell me if you can get an API key.

### Option C — Per-category chats (best quality, some manual work)
Paste the block, then `generate #1 … #N` within each category chat. ~5 chats, a few words each. Slowest typing but highest control and full resolution. This is the default workflow above.

**Recommendation:** Option A for a quick style preview → then **Option B** to mass-produce the finals.
