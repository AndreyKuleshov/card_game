# Оверлей завершения боя + сундук с наградой

## Цель
Не уходить на отдельный экран награды (сейчас `pushReplacement` → `RewardScreen`,
из-за чего экран «сдвигается»). Показывать исход боя **оверлеем поверх дуэли**.
На победе — сундук, который открывается по клику и из него вываливаются кристаллы и
козырь (если выпал).

## Поток
- Бой заканчивается (`outcome != ongoing`) → вместо навигации ставим состояние
  оверлея: `_finished=true`, `_won`, `_reward` (из `computeDuelReward`), `_chestOpen=false`.
- Оверлей — слой в `Stack` поверх сцены дуэли: затемнённый фон + контент по центру.

### Победа
- Заголовок «Победа!». Закрытый сундук (`chest_closed.png`) + подсказка
  «Нажми, чтобы открыть».
- Тап по сундуку → `_chestOpen=true`: картинка меняется на `chest_open.png`,
  анимированно появляются кристаллы (`💎 N` / иконка) и карта-козырь (если
  `reward.trumpGranted != null`). В этот момент награды **начисляются**:
  `addCrystals`, `grantCard`, `unlockNextNode`.
- Кнопка «В королевство» → `Navigator.popUntil((r) => r.isFirst)`.

### Поражение
- Оверлей 💀 «Поражение» + кнопка «В королевство». Сундука нет, наград нет.

## Ассеты
