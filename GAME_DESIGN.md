# Дизайн-документ: фэнтези карточная игра (рабочее название TBD)

**Дата:** 2026-06-22
**Платформа:** Android (стек — Flutter)
**Объём первой версии:** Вертикальный срез (MVP) — один полный игровой цикл
**Тон:** ламповое фэнтези с юмором, шутливые названия карт (не пафосное дарк-фэнтези)

---

## 1. Концепция

Карточная фэнтези-игра, где игрок:
1. Побеждает в **дуэлях** (сравнение карт), снося ХП замка соперника.
2. За победы получает **ресурсы** и развивает своё **королевство**.
3. Развитие королевства усиливает колоду и замок, а также производит ресурсы.
4. Добывает всё более мощные карты — **козыри** — тремя путями, чтобы побеждать более сильных противников и дойти до босса.

Базовая петля:
> дуэль → ресурсы → улучшение королевства (сильнее карты / крепче замок / больше дохода) + новые козыри → более сильные противники → ... → босс → уникальный козырь

---

## 2. Карты и стихии

**Поля карты:**
- `name` — шутливое имя
- `element` — стихия: 🔥 Огонь / 🌿 Природа / 💧 Вода
- `power` — базовая сила, 1–10
- `rarity` — обычная / редкая / **козырь**
- `ability` — только у козырей (см. §5)

**Стихийный круг:** Огонь → Природа → Вода → Огонь (каждая бьёт следующую).
При столкновении, если стихия в преимуществе над стихией соперника, сила получает **+3** (настраиваемый параметр). Нет преимущества → сравнение по чистой силе.
Следствие: слабая карта может победить сильную за счёт стихийного преимущества — отсюда тактика выбора хода.

**Стартовый набор обычных карт (по 4 на стихию для среза; сила распределяется 2–8 при балансировке):**

| Стихия | Карты |
|---|---|
| 🔥 Огонь | Горячий Олень, Желанный Петушок, Горелый Пирожок, Перл Феникса |
| 🌿 Природа | Боевой Кабачок, Пьяный Лесник, Гриб-Качок, Боевой Ёж |
| 💧 Вода | Сердитая Медуза, Капитан Лужа, Бобёр-Сантехник, Ледяной Пельмень |

Плюс 3–4 козыря (см. §5).

---

## 3. Ход дуэли

1. У обоих игроков **колода** (для среза — 12 карт из коллекции) и **рука** на 4 карты.
2. **Раунд:** оба выбирают карту из руки и вскрывают одновременно.
3. **Эффективная сила** = базовая сила + стихийный бонус (+3, если есть преимущество) + бонусы зданий королевства + эффекты способностей.
4. Кто сильнее — наносит **разницу сил** урона по ХП замка соперника. Ничья — урона нет.
5. Добор руки до 4 карт, следующий раунд.
6. Партия длится, пока чей-то **замок (старт 30 ХП)** не обнулится. Если у обоих кончились карты — побеждает игрок с большим ХП замка.

Целевая длительность партии: ~3–5 минут (мобильный формат).

Пример: Боевой Кабачок 🌿 (сила 5) против Уголька 🔥 (сила 7). Огонь бьёт Природу → у огненной карты +3 = 10 против 5. Победа огня, −5 по замку соперника.

---

## 4. Королевство и экономика

**Ресурс (один для среза):** 💎 **Кристаллы**. Источники: награда за победу в дуэли + пассивная/победная добыча с шахты.

**Здания (гибрид: боевые + производство).** Старт — **ур.0 (не построено, 0 бонуса)**; игрок строит и качает до ур.3 за кристаллы (стоимость: построить 5, ур.1→2 = 10, ур.2→3 = 25).

| Здание | Тип | Эффект (ур.0 → ур.3) |
|---|---|---|
| 🏹 Казарма | боевое | 0 → +3 к силе карт выбранной стихии в дуэли (+1 за уровень) |
| 🧱 Стена | боевое | 0 → +15 к стартовому ХП замка (+5 за уровень) |
| ⛏️ Шахта | производство | 0 → +6 кристаллов за победу (+2 за уровень) |

Для среза королевство — это один экран со списком зданий и кнопками «Улучшить» (без расстановки на карте/изометрии — отложено).

Крафт козыря выполняется в **Казарме на ур.3** за крупную сумму кристаллов (см. §5).

## 5. Козыри и прогресс

**Козырь** — редкая карта с высокой силой (8–10) и одной простой способностью.

**Способности (для среза):**
- *Стихийный сдвиг* — игнорирует стихийный бонус соперника в этом раунде.
- *Двойной удар* — при победе в раунде наносит ×1.5 урона.
- *Щит* — при проигрыше раунда замок не получает урон.

**Три способа добычи (комбинация):**
1. 🏆 **Босс-трофей:** один босс в срезе. Победа → «Король-Тыква» (`trump_pumpkin_king`) 🌿.
2. 🔨 **Крафт:** создаётся в Казарме ур.3 за кристаллы → «Лавовый Котик» (`trump_lava_cat`) 🔥 (предсказуемый путь).
3. 🎁 **Сундук:** за победу в обычной дуэли ~30% шанс → «Морозная Бабуля» (`trump_frost_granny`) 💧 (рандомный путь).
4. 🐣 **Стартовый:** выдаётся при первом запуске → «Дракоша-Стартер» (`trump_starter_drake`).

Итого в срезе **4 козыря**: по одному на каждый способ добычи. Покрывает все механики без раздувания контента.

**Карта мира (срез):** линейная цепочка узлов:
`Тренировка (ИИ-болванчик) → Противник 1 → Противник 2 → БОСС`.
Победа над узлом открывает следующий.

## 6. Техническая архитектура (Flutter)

Слоистая: чистый движок отделён от UI, онлайн добавляется без переписывания логики.

- **`engine/` — чистый Dart, без Flutter.** `Card`, `Deck`, `ElementRules` (круг стихий + бонус), `DuelEngine` (раунды, расчёт урона, условие победы), `Kingdom`/`Economy` (здания, кристаллы), `AbilityResolver`. Покрывается юнит-тестами.
- **`PlayerController` — интерфейс хода.** Реализации: `HumanController`, `AiController` (эвристика). Позже — `RemoteController` для PvP; движок не меняется.
- **Состояние:** Riverpod (провайдеры для коллекции, королевства, текущей дуэли).
- **Сохранение:** локально, JSON через Hive/shared_preferences (коллекция, ресурсы, уровни зданий, открытые узлы).
- **Данные карт:** JSON-ассет (`cards.json`) — добавление карт без перекомпиляции логики.
- **UI-экраны:** Королевство, Колода, Карта мира, Дуэль, Награда/Сундук.

```
lib/
  engine/        # правила (pure Dart, тестируемо)
  models/        # Card, Building, GameState
  controllers/   # Human/Ai/(Remote)
  state/         # Riverpod-провайдеры
  ui/            # экраны и виджеты
  data/          # загрузка cards.json
assets/cards.json
test/            # юнит-тесты движка
```

## 7. Объём среза (что входит / что отложено)

**Входит в срез:**
- 12 обычных карт + 3–4 козыря, 3 стихии.
- Полная дуэль против ИИ с ХП замка и расчётом урона.
- Королевство: 3 здания × 3 уровня, ресурс Кристаллы.
- Все три способа добычи козырей (босс/крафт/сундук) — по минимуму контента.
- Линейная карта мира: 3 узла + босс.
- Локальное сохранение.

**Отложено (после среза):**
- Онлайн PvP (архитектурный задел есть).
- Второй ресурс (напр. 🪵 Дерево), больше зданий и расстановка на карте.
- Регионы, больше карт/козырей/способностей.
- Монетизация, метапрогресс аккаунта.
